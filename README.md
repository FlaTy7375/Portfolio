# Portfolio Website

Мое портфолио с интерактивными 3D элементами и анимациями.

## Технологии

- **React 18** - UI библиотека
- **Vite** - Сборщик и dev-сервер
- **React Three Fiber** - 3D графика
- **Styled Components** - CSS-in-JS стилизация
- **Three.js** - 3D библиотека

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/public`

## Предпросмотр продакшен сборки

```bash
npm run preview
```

## Структура проекта

```
client/
  ├── public/          # Статические файлы
  ├── src/
  │   ├── components/  # React компоненты
  │   │   ├── blocks/  # Секции сайта
  │   │   └── layout/  # Компоненты layout
  │   └── styles/      # Глобальные стили
  └── index.html
```

## Особенности

- Адаптивный дизайн для всех устройств
- Интерактивная 3D модель в Hero секции
- Анимированное облако навыков
- Портфолио проектов с hover эффектами
- Плавные переходы и анимации
- Оптимизация производительности для Safari
